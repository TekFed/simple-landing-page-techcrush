# Simple Landing Page — TechCrush

A small static landing page built with HTML and CSS, featuring:

- A responsive hero section using `bg_logo.jpg`
- Section reveal animations on scroll
- Light and dark mode toggle with preference persistence

## Files

- `index.html` — page markup
- `styles.css` — page styling and theme variables
- `bg_logo.jpg` — hero image asset

## Usage

1. Open `index.html` in your browser.
2. Scroll down to see content reveal animations.
3. Use the theme toggle button to switch between light and dark mode.

## Notes

This is a self-contained static project and does not require a server or build tools.
